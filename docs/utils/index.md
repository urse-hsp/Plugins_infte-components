# utils

- @infte/utils 说明文档
- 下载 npm i @infte/utils

## localStorage

- 本地储存
- 类型：(key: string, value?: any): any

## byteLength

- 判断字节长度
- 类型：(str: string): number

## JudgmentBytes

- 限制字符串输入长度 字节
- 类型：(msg: string, num: number): any

## substringLength

- 截取字符串多少个字符：suffix:后缀
- 类型：(str: string, len = 12, suffix = '...')

## isUrl

- 判断 url
- 类型：(str: string): boolean

## getFileType

- 判断资源类型
- 类型：(name: string): any
