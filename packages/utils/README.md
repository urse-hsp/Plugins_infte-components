# @infte/utils 说明文档

## localStorage

- 类型：(key: string, value?: any): any
- 本地储存

## byteLength

- 类型：(str: string): number
- 判断字节长度

## JudgmentBytes

- 类型：(msg: string, num: number): any
- 限制字符串输入长度 字节

## substringLength

- 类型：(str: string, len = 12, suffix = '...')
- 截取字符串多少个字符：suffix:后缀

## isUrl

- 类型：(str: string): boolean
- 判断 url

## getFileType

- 类型：(name: string): any
- 判断资源类型
